## Run on Computer/VPS
 - Install Python and Git

**For Windows**
```
winget install python
winget install Git.Git
```

**For Linux(APT)**
```
sudo apt-get update && sudo apt-get install -y python git pip
```

**For MAC (HomeBrew)**
```
brew install python git
```
 - Clone Repository
 - Install Requirements
```
pip install -r requirements.txt
```
 - Run Python Program
```
python main.py
```
